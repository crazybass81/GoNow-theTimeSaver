### A-1. 페널티 수익 배분 정책

**질문**: 사용자가 지각하여 차감된 페널티 금액(예: 10,000원)을 어떻게 처리할 것인가?

**선택지**:
```
[ ] Option A: 100% 회사 수익으로 계상
    장점: 수익 극대화
    단점: 도덕적 비난 리스크, "앱이 일부러 늦게 알림" 의심
    예상 월 수익 (사용자 1,000명 기준): 500만 원

[ ] Option B: 일부 기부 (30~50%)
    배분: 회사 50% + 자선단체 50%
    장점: 도덕적 명분 확보, 마케팅 포인트
    단점: 수익 감소
    예상 월 수익: 250만 원

[ ] Option C: 플랫폼 수수료만 (20%)
    배분: 회사 20% + 나머지는 기부 또는 보험 펀드
    장점: 공정성 인정, 앱스토어 심사 유리
    단점: 최저 수익 모델
    예상 월 수익: 100만 원

[ ] Option D: 사용자 풀 (Jackpot)
    설명: 성공자들끼리 실패자 페널티 나눠가짐
    장점: 게임화 강화, 커뮤니티 형성
    단점: 사행성 규제 저촉 가능성 (⚠️ 법률 검토 필수)
```
